⚠ Données FAQ non chargées

Pour activer cet onglet, exporte le rapport HubSpot et place le CSV dans le dossier Lighthouse :
  1. Ouvrir le rapport HubSpot : → FAQ : comparaison performances
  2. Bouton "Exporter" en haut à droite → format CSV → télécharger
  3. Renommer le fichier faq-comparaison-performances.csv
  4. Le placer dans ~/Documents/lighthouse_setup/
  5. Au prochain refresh (auto à HH:00 ou python3 refresh.py), les données apparaîtront ici

À refaire chaque semaine pour avoir les chiffres à jour. Une intégration automatique via API HubSpot Reports nécessiterait un scope Hub Analytics avancé non activé sur ton compte.
